# BookVerse Platform

Modern fullstack book platform focused on responsive UI, interactive customization, and backend-driven functionality.

## Features

- Responsive modern interface
- Interactive book showcase
- Customizable book covers
- Pricing and rental system
- User authentication system
- CMS-style book management
- Demo order functionality
- Smooth animations and transitions
- Mobile-friendly layout

## Technologies

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js

## Functionality

- Browse and explore books
- Rent or purchase books
- Customize book appearance
- Manage books through CMS functionality
- User account system
- Demo order submission with email notifications
- Dynamic frontend and backend interaction

## Goals

This project was created to improve fullstack development skills by combining modern frontend design with backend functionality, API handling, authentication systems, and CMS-style content management.

## Status

Currently in active development.
